# 🎬 CineVault

CineVault is a modern movie discovery web app built using React and TMDB API.  
It allows users to explore trending movies, view detailed information, watch trailers, and save their favorite movies.

---

## 🚀 Features

- 🔍 Search movies in real time
- 🎬 Trending movies section
- 📄 Detailed movie pages with backdrop, rating, and metadata
- 🎥 Watch trailers (YouTube integration)
- ❤️ Add movies to watchlist (localStorage)
- 🎞 Similar movies suggestions
- 📱 Responsive UI design

---

## 🛠 Tech Stack

- React.js
- React Router DOM
- TMDB API
- CSS3
- JavaScript (ES6+)
